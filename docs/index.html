Wishgranter Rest API (1.0)

Download OpenAPI specification:

This describes a restful API for a web-based music player. This is part of the Wishgranter project.

playlistItems

Everything about your playlist items

List playlist items.

Returns a list of playlists items.

Authorizations:
cookieAuthentication
path Parameters
userId
required
string
Example: valery

The user's unique id in system.

query Parameters
itemsPerPage
integer
Example: itemsPerPage=20

The maximum number of items to present per page.

page
integer
Example: page=1

The page we are aiming for.

playlistId
Array of strings
Example: playlistId=samurais-best&playlistId=night-city-summer-hits

The ids of playlists to search for items. If empty, all playlists in the user's collection will be subject to the search.

title
string
Example: title=Black Dog

Search parameter to match against the items' title.

artist
string
Example: artist=Johnny Silverhand

Search parameter to match against the items' artist(s).

genre
string
Example: genre=Rock

Search parameter to match against the items' genre(s).

soundtrack
string
Example: soundtrack=Rock

Search parameter to match against the items' soundtrack(s).

orderBy
string
Example: orderBy=RAND(seed-goes-here)

Property to order results by.
